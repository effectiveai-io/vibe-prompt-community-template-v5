기본 컴포넌트

Button

다양한 스타일과 상태를 지원하는 버튼 컴포넌트

<Button variant="default">Default</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="outline">Outline</Button>
<Button variant="destructive">Destructive</Button>

Input

텍스트 입력을 위한 기본 입력 필드

<Input placeholder="Enter text..." />
<Input type="password" placeholder="Password" />
<Input disabled placeholder="Disabled input" />

Textarea

여러 줄 텍스트 입력을 위한 텍스트 영역

<Textarea placeholder="Enter multiple lines..." />

Label

폼 요소와 연결되는 레이블 컴포넌트

<Label htmlFor="email">Email Address</Label>
<Input id="email" type="email" />

Checkbox

선택 가능한 체크박스 컴포넌트

<Checkbox id="terms" />
<Label htmlFor="terms">Accept terms and conditions</Label>

Radio Group

단일 선택을 위한 라디오 버튼 그룹

<RadioGroup defaultValue="option1">
  <RadioGroupItem value="option1" id="option1" />
  <Label htmlFor="option1">Option 1</Label>
  <RadioGroupItem value="option2" id="option2" />
  <Label htmlFor="option2">Option 2</Label>
</RadioGroup>

Select

드롭다운 선택 컴포넌트

<Select>
  <SelectTrigger>
    <SelectValue placeholder="Select an option" />
  </SelectTrigger>
  <SelectContent>
    <SelectItem value="option1">Option 1</SelectItem>
    <SelectItem value="option2">Option 2</SelectItem>
  </SelectContent>
</Select>

Switch

토글 스위치 컴포넌트

<Switch />
<Switch checked />
<Switch disabled />

Slider

값 범위 선택을 위한 슬라이더

<Slider defaultValue={[50]} max={100} step={1} />

    <TabsTrigger value="tab1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab2">Tab 2</TabsTrigger>
  </TabsList>
  <TabsContent value="tab1">Content 1</TabsContent>
  <TabsContent value="tab2">Content 2</TabsContent>
</Tabs>

Breadcrumb

현재 위치를 표시하는 브레드크럼

<Breadcrumb>
  <BreadcrumbList>
    <BreadcrumbItem>
      <BreadcrumbLink href="/">Home</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbPage>Current Page</BreadcrumbPage>
    </BreadcrumbItem>
  </BreadcrumbList>
</Breadcrumb>
Home / Category / Current Page

      <PaginationPrevious href="#" />
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="#">1</PaginationLink>
    </PaginationItem>
    <PaginationItem>
      <PaginationNext href="#" />
    </PaginationItem>
  </PaginationContent>
</Pagination>

피드백

Toast

일시적인 알림 메시지

import { useToast } from "@/hooks/use-toast"

const { toast } = useToast()

toast({
  title: "Success!",
  description: "Your changes have been saved.",
})

Success!

Your changes have been saved.

Alert

중요한 정보를 표시하는 알림

<Alert>
  <AlertCircle className="h-4 w-4" />
  <AlertTitle>Heads up!</AlertTitle>
  <AlertDescription>
    You can add components to your app using the cli.
  </AlertDescription>
</Alert>
Heads up! You can add components to your app using the cli.
Success! Your operation was completed successfully.
Warning! Please check your input before proceeding.
Error! Something went wrong. Please try again.

Accordion

접을 수 있는 콘텐츠 섹션

<Accordion type="single" collapsible>
  <AccordionItem value="item-1">
    <AccordionTrigger>Is it accessible?</AccordionTrigger>
    <AccordionContent>
      Yes. It adheres to the WAI-ARIA design pattern.
    </AccordionContent>
  </AccordionItem>
</Accordion>
Is it accessible?
Yes. It adheres to the WAI-ARIA design pattern.
Is it styled?
Yes. It comes with default styles that match your design system.

Form

React Hook Form과 통합된 폼 컴포넌트

<Form {...form}>
  <form onSubmit={form.handleSubmit(onSubmit)}>
    <FormField
      control={form.control}
      name="email"
      render={({ field }) => (
        <FormItem>
          <FormLabel>Email</FormLabel>
          <FormControl>
            <Input placeholder="Enter email" {...field} />
          </FormControl>
          <FormMessage />
        </FormItem>
      )}
    />
  </form>
</Form>
